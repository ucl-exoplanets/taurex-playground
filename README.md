# TauREx-Playground

A collection of notebooks, scripts and tools to help you get started with TauREx.

## Colab Notebooks

- [TauREx-Modelling](https://colab.research.google.com/drive/1RppsquYSrQeHdmjDrwgjrO7DuyQTgJ6O?usp=sharing)
    - A guide to get started with forward modelling with taurex.
- [TauREx-Retrievals](https://colab.research.google.com/drive/1u3GpTPbXA6y9SG4-3mVljQVrl7Dh_Brr?usp=sharing)
    - A guide to performing retrievals with taurex.
- [TauREx-CuPy](https://colab.research.google.com/drive/1wGm1FiJfPQOwVqNcmEj8s3ZKxm9PvZtm?usp=sharing)
    - A guide to using CuPy with TauREx for faster retrievals.
